Foreign Exchange Rates

This is a web application created using Flutter. This web app fetches the latest exchange rates of currencies from the European Central Bank's API. Users can enter their currency code and it will fetch the exchanges rates for 5 other currencies (USD,EUR,JPY,CNY and SGD). A currency converter is also provided and will convert currencies based on the exchange rates provided by the API.
